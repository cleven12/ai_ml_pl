## ICT Club — Intro to AI/ML Full Curriculum

**Format:** Evening sessions, 2hrs each, 19:00–21:00 EAT <br>
**Pace:** No rushing at ALL <br>
**Tools:** Anaconda (Python + Jupyter), no cloud accounts needed

---

### Phase 1 — Foundations (No Code)

**Session 1 — What is AI?**
What is AI, ML, Deep Learning — differences and relationships. History from rule-based systems to modern AI. How AI actually works at a conceptual level (pattern recognition, not magic).

**Session 2 — Data is Everything**
What is data, why it matters, types of data (structured/unstructured). How machines "see" data. What makes good vs bad data. Bias starts here — garbage in, garbage out. 
<br>Activity: Explore real dataset together (CSV of something local — weather, population, anything).

**Session 3 — Types of Machine Learning**
Supervised, unsupervised, reinforcement learning — with real analogies. Classification vs regression vs clustering. No code yet, just building strong mental models.
<br>Examples : spam filters, recommendation systems, fraud detection.

---

### Phase 2 — Environment Setup + First Code

**Session 4 — Setting Up Our Workshop**
Install Anaconda. Tour of Jupyter Notebook. Python crash course for absolute beginners — variables, lists, loops, functions. No ML yet, just getting comfortable in the environment.

**Session 5 — Python for Data**
Intro to NumPy and Pandas. Loading a CSV, exploring it, basic stats. Visualizing data with Matplotlib — bar charts, histograms. 
<br>Activity: load a real Tanzanian dataset (TANROADS, weather, anything available).

**Session 6 — Your First ML Model**
Intro to scikit-learn. Train/test split concept. Build a simple classifier (e.g. predict if something is A or B). Evaluate it — accuracy, confusion matrix. They will see the full pipeline for the first time.

---

### Phase 3 — Core ML Algorithms

**Session 7 — Linear & Logistic Regression**
Linear regression: predicting a number (e.g. house price). 
<br> Logistic regression: predicting a category (yes/no). Math intuition only — no heavy formulas. Code both in sklearn.

**Session 8 — Decision Trees & Random Forests**
How a decision tree thinks. Overfitting problem and why it matters. Random forest as the fix. Feature importance — which inputs matter most? Hands-on with a dataset.

**Session 9 — Clustering & Unsupervised Learning**
K-Means clustering. When you don't have labels. 
<br> Real use case: customer segmentation, anomaly detection. Visualizing clusters with matplotlib.

**Session 10 — Model Evaluation & Improvement**
Cross-validation. Precision, recall, F1 score — when accuracy lies. Hyperparameter tuning basics. How to know if your model is actually good.

---

### Phase 4 — Neural Networks & Deep Learning

**Session 11 — Introduction to Neural Networks**
Biological neuron → artificial neuron analogy. 
<br> Layers: input, hidden, output. Forward pass concept. No code yet — draw it, understand it deeply first.

**Session 12 — Building a Neural Network**
Intro to TensorFlow/Keras (also in Anaconda). Build a simple feedforward network. Train it on a dataset. Watch loss go down — understand what's happening.

**Session 13 — Computer Vision Basics**
How machines see images (pixels → arrays). Convolutional Neural Networks (CNN) concept. Use a pretrained model to classify images. OpenCV basics — load, display, transform an image.

**Session 14 — Natural Language Processing Basics**
Text as data — tokenization, vectorization. Sentiment analysis with a simple model. Why Swahili and Bantu languages are underrepresented. 
<br>Demo: run a language detection tool, see it fail on Swahili, discuss why.

---

### Phase 5 — Applied AI & Real Projects

**Session 15 — Working with AI APIs**
What is an API, how to call one. Use Claude or Gemini API (free tier). Build a simple question-answering script in Python. Bridge between "using AI" and "building with AI."

**Session 16 — Mini Project: Build Something Real**
Each member or small group picks a local problem. 
<br>Examples: crop disease detection, Swahili text classifier, tourism recommendation, school result predictor. Define the problem, find/create data, plan the model.

**Session 17 — Project Build Session**
Hands-on coding session. Groups work on their mini projects. Debug together, share blockers.

**Session 18 — Project Presentations + What's Next**
Each group presents their project — even if incomplete. Discuss what worked, what didn't, what they'd do differently. 
<br> Career paths: ML engineer, data scientist, AI researcher, AI product builder. 
<br> Resources: Kaggle, fast.ai, Papers With Code, AfricaNLP. 
<br> Close:  everyone to keep going.

---

**Tools across all sessions:**

- TensorFlow/Keras (installed via conda when we reach Session 12)
